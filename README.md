# Insta_Book
🚀 Excited to share InstantBook – A Full-Stack Event Booking Platform 🎟️

InstantBook allows users to discover events, book free or paid tickets, and manage their bookings seamlessly — while organizers can create, publish, and track events through a dedicated dashboard.

🌐 Visit here : https://lnkd.in/gTJzjnQ3

🔹 Key Features

• Secure Authentication using JWT + HTTP-only cookies
• Real-time Seat Tracking with Sold-Out Detection
• Razorpay Payment Integration for paid bookings
• Organizer Dashboard & Ticket Management
• Fully Responsive Modern UI
